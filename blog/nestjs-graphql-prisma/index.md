# NestJS + Prisma + GraphQL 开发入门

## NestJS

```shell
# 全局安装 NestJS
yarn i -g @nestjs/cli

# 创建 NestJS 项目
nest new project-name
```

## Prisma

```shell
yarn add prisma --dev
yarn prisma init
```
